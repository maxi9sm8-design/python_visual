Fundamentos de Python - Guia de Aprendizaje

Este repositorio contiene una serie de ejercicios practicos diseñados para dominar la logica de programacion y las estructuras de datos en Python. Cada archivo incluye ejemplos comentados que explican el funcionamiento de cada concepto.

Contenido del Repositorio

El proyecto esta organizado en los siguientes modulos:

Entrada y Salida de Datos

print.py: Uso de la funcion de salida, impresion de numeros, variables y caracteres especiales.

input.py: Captura de datos del usuario y conversion de tipos para realizar operaciones.

Estructuras de Datos

listas.py: Colecciones mutables, acceso por indice, listas anidadas e iteracion con bucles.

tuplas.py: Uso de colecciones inmutables, acceso a elementos y estructuras anidadas.

diccionarios.py: Almacenamiento mediante pares clave-valor, mutabilidad e iteracion.

Control de Flujo

if elif else.py: Toma de decisiones logica basada en condiciones verdaderas o falsas.

match case.py: Estructura de comparacion de patrones para un codigo mas legible.

for.py: Bucles determinados para iterar rangos o listas, incluyendo break, continue y pass.

while.py: Bucles basados en condiciones, validacion de datos y el uso de la clausula else.

Conceptos Avanzados

anidacion.py: Combinacion de estructuras de control para resolver problemas complejos sin repetir codigo.
